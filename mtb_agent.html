🔵 Garmin Connect

Se usa la librería garminconnect (Python). Las credenciales solo se usan localmente para autenticar contra Garmin Connect — no se almacenan ni envían a ningún servidor externo.

Para extraer MTB Dynamics (flow, grit, jumps) el agente descarga los archivos .FIT y los parsea localmente.

📄 Google Doc · Setup Sheet

El documento debe estar compartido como "Cualquiera con el enlace puede ver". El agente extrae los valores de setup actuales para correlacionarlos con el rendimiento de cada salida.

⚙ Script de instalación Python

Ejecuta esto en tu terminal Mac para instalar las dependencias del agente: 

pip3 install garminconnect requests python-dotenv fitparse
